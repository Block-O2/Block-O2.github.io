Ongoing

Adaptive Traction Control with Constrained MPC

Supervised by Prof. Fangxun Zhong

Developing and evaluating adaptive model predictive control for a compliant traction task under uncertain dynamics. The current system combines constrained NMPC, state estimation, and online parameter identification, with explicit force, deformation, velocity, and acceleration considerations.

Research repository ↗

Current scope
Spring2D simulation, planner–tracker NMPC, UKF-based estimation, and windowed parameter identification
Development path
Compliant single-link model → articulated systems → MuJoCo → hardware

Sagittarius Semantic Grasping

Language-guided semantic grasping pipeline deployed on a physical Sagittarius robot arm.

MT3 Dynamic Alignment Extension

Extended MT3's static alignment pipeline toward continuous tracking of moving objects using Kalman filtering and SE(2) pose estimation.

  • Designed the dynamic tracking architecture and validated it in synthetic PyBullet experiments.
  • Built a 66-test verification suite covering geometry, filtering, tracking behavior, and failure cases.
Python · PyBullet · Kalman Filter · SE(2) · Unit Testing
GitHub ↗
03

ABS Quadruped Robot Sensor Debugging

Debugged the sensor-data chain for a Go2 quadruped control system, aligning IMU, joint-state, foot-contact, and LiDAR observations.

  • Diagnosed gait instability and observation inconsistencies across multiple sensor streams.
  • Documented the troubleshooting process and verified the corrected data flow.
ROS · IMU · LiDAR · Sensor Fusion · System Debugging
04

NavRL Environment Replication

Reproduced a navigation training pipeline in Isaac Sim and verified the policy-training workflow.

  • Configured the simulation and training environment and resolved library and dependency conflicts.
  • Verified the training chain in simulation; real-robot deployment was not attempted.
Isaac Sim · Reinforcement Learning · Python · C++

05

RL Inverted Pendulum Control

Modeled and controlled an inverted pendulum in MATLAB/Simulink, comparing Q-learning with classical feedback control.

  • Defined the state space, action space, and reward structure for swing-up and balancing tasks.
  • Evaluated Q-learning and PD control under the same simulated system.
MATLAB · Simulink · Q-Learning · PD Control · State-Space Modeling
